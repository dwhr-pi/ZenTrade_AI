Balkendiagramm - Finanzanalyse

Beschreibung

Balkendiagramme stellen Datenpunkte als horizontal oder vertikal positionierte Balken dar. Sie werden häufig verwendet, um Kennzahlen über verschiedene Kategorien hinweg zu vergleichen, wie z.B. Einnahmen nach Produktlinien oder Ausgaben nach Abteilungen. Dieses Beispiel zeigt einen Vergleich der Quartalsumsätze verschiedener Geschäftsbereiche.

Verwendete Variablen

Variable Beschreibung Datentyp
labels Kategorien auf der X-Achse (Geschäftsbereiche) Array von Strings
datasets[].label Bezeichnung des Datensatzes (Quartal) String
datasets[].data Umsatzwerte für jede Kategorie Array von Zahlen
datasets[].backgroundColor Hintergrundfarbe der Balken String oder Array von Strings (CSS-Farbwerte)
datasets[].borderColor Randfarbe der Balken String oder Array von Strings (CSS-Farbwerte)
datasets[].borderWidth Breite des Balkenrands in Pixeln Zahl
