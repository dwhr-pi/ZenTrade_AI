# ZenTrade AI React Components

Dieses Paket enthält alle bereitgestellten ZenTrade-AI-SVGs als React/TypeScript-Komponenten.

## Anpassung

`zentrade-ai.svg` wurde optisch verkleinert und auf eine 512x512-ViewBox normalisiert, damit es gegenüber den anderen Icons nicht mehr zu groß wirkt.

## Nutzung

```tsx
import { ZentradeAiIcon, WalletIcon, AutoTradingIcon } from "./react-components";

export default function Example() {
  return (
    <div className="flex gap-4">
      <ZentradeAiIcon size={96} />
      <WalletIcon size={96} />
      <AutoTradingIcon size={96} />
    </div>
  );
}
```
